About the developer

About the Developer

Hi, I'm Al Shahriar Sayon — the developer behind Bean Run, a personal indie game project I built out of curiosity and a genuine love for games. I wrote the core game logic and the complete Python / Pygame source code myself, shaping how the game plays, feels, and comes together.

Along the way, I used OpenAI ChatGPT and Claude Code as helping hands to sharpen the application architecture, the launcher and installer, this website, and the overall UI/UX — as well as optimization, debugging, and deployment. Bean Run was built for three simple reasons: to learn, to create, and to share something with the community.

I'm currently a 2nd Semester student in the Computer Science & Technology (CST) department at Dhaka Polytechnic Institute. My goal is to keep growing in software and game development by building real projects — one idea at a time.
