About the game

Built for fun, made to learn

Bean Run is a lovingly crafted endless arcade runner written in Python using the Pygame library. It started as a fun weekend project and grew into a fast, addictive little game about reflexes, timing, and chasing that next high score.

The entire project is open source and intentionally beginner friendly. Every mechanic — the movement, the obstacles, the scoring, the music — is written in clean, readable code so you can learn from it, tweak it, and make it your own.
